Mesure de la performance

Sur cette page

  • Déterminer les indicateurs clés
  • Obtenir des données
  • Analyser et rapporter
  • Améliorer

La mesure de la performance est un moyen de recueillir et d'analyser des données pour déterminer si un produit atteint son objectif ou ses buts.

Cycle de mesure de la performance:

Déterminer les indicateurs clés

Dans cette section du cycle, vous :

  • décidez quoi mesurer
  • partez de votre feuille de route de produit et clarifiez les objectifs du produit
  • déterminez à quoi ressemble le succès : ce que vous voulez que les visiteurs fassent avec votre contenu, et quelles sont les principales tâches que vous voulez qu'ils accomplissent
  • faites participer votre analyste web dès le début du processus
    • À ce stade, ils peuvent vous aider à définir des objectifs et à déterminer la meilleure façon de les mesurer.
    • Une analyse appropriée et des renseignements clairs plus tard dans le cycle nécessitent de définir des mesures qui décrivent clairement les objectifs souhaités à cette étape.

Ces décisions déterminent les données qui sont collectées et mesurées.

Obtenir des données

C'est la phase où nous commençons à collecter des données.

Voici quelques-unes des mesures d'analyse Web que nous pouvons mesurer.

Nombre de visiteurs

Combien de visiteurs sont venus sur votre page

Nombre de visites

Combien de fois des gens sont allées sur votre page

Références

Comment les gens sont-ils arrivés sur votre page (moteur de recherche, site partenaire, e-mail de partenaire, etc.)

Temps engagé sur la page

Combien de temps le visiteur a-t-il activement consulté la page

Défilement de la page

Jusqu'où le visiteur est-il allé sur la page

Minuteur de visualisation d'élément

Combien de temps les visiteurs ont-ils visualisé les éléments clés de la page (cartes, graphiques, tableaux, etc.)

Sélections de liste déroulante

Avec quelle fréquence les sélections des menus déroulants ont-elles été utilisées sur la page

Clics de lien sortant

La fréquence à laquelle les liens dirigeant les visiteurs vers des pages web externes ont été cliqués

Texte copié

Avec quelle fréquence et quel texte spécifique sur la page a été copié dans le presse-papiers

Enregistrements de session

Vidéo de la façon dont le visiteur a visualisé votre page, montrant où ils ont cliqué, les mouvements de la souris, les défilements. C'est comme si vous étiez derrière quelqu'un pendant qu'il interagit avec votre page. Ces enregistrements peuvent aider à évaluer si les visiteurs interagissent avec le produit comme nous l'avons prévu, à découvrir les éléments qui fonctionnent et à identifier les domaines à améliorer.

Analyser et rapporter

Dans cette partie du cycle, vous :

  • organisez les données en rapports lisibles pour surveiller la performance et la réussite
  • extrayez des renseignements significatifs sur la façon d'améliorer la performance
  • utilisez l'analytique pour prendre des décisions basées sur les preuves
  • essayez de déterminer les modifications à apporter à votre contenu ou design pour les rendre plus efficace

Si vous n'avez pas atteint vos objectifs, ce n'est pas grave. Appliquez les leçons apprises à la prochaine version ou mise à jour.

Améliorer

Dans cette partie du cycle, vous :

  • utilisez les renseignements tirés de l'analytique comme déclencheur pour démarrer un processus d'amélioration
  • ajoutez d'autres sources de données pour comprendre comment améliorer le produit

C'est un processus itératif : continuez à surveiller les données et à améliorer le produit

Allez à Amélioration continue pour plus de détails.
